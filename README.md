# Cursor Gremlins

A silly, dependency-free browser toy where cursed little blobs invade your page and react to your cursor like tiny unpaid interns.

## Features

- Click anywhere to summon gremlins with random names, personalities, colors, behaviors, and bad habits.
- Gremlins chase the cursor, avoid it, bounce around, duplicate, and nibble the UI.
- Chaos Meter with a locally saved high chaos score.
- Panic, reset, and summon 10 buttons.
- Random events like gravity flips, disco mode, corporate tiny tie mode, cursed cursor, runaway buttons, huge gremlins, and mayor elections.
- URL seed support with `?seed=your-chaos-seed` for shareable weirdness.
- Responsive, smooth, cute, cursed, and made with plain HTML, CSS, and vanilla JavaScript.

## Run locally

Open `index.html` in your browser. That is it. No install. No build step. No dependency goblin.

## Deploy on GitHub Pages

1. Push these files to a GitHub repository.
2. Go to **Settings → Pages**.
3. Choose **Deploy from a branch**.
4. Select the `main` branch and `/root` folder.
5. Save, wait a minute, and let the gremlins become public infrastructure.

## Dependency-free

This project uses no frameworks, no packages, no backend, and absolutely no sensible adult supervision.
